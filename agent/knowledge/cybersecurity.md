# ARES Knowledge Base

Este directorio contiene notas técnicas para que ARES razone sobre ciberseguridad con un enfoque mixto, con prioridad en red team pero siempre incluyendo defensa, detección y mitigación cuando aplique.

## Convención de cada nota

Cada archivo debe seguir una estructura consistente:
- Resumen del tema.
- Conceptos clave.
- Comandos y sintaxis.
- Ejemplo práctico en terminal.
- Escenario real de ataque y defensa.
- Detección y mitigación.
- Herramientas relacionadas.
- Referencias útiles.

## Notas principales

- [linux.md](linux.md): permisos, procesos, red, hardening y administración.
- [windows.md](windows.md): privilegios, servicios, PowerShell, logs y defensa.
- [python.md](python.md): Python aplicado a automatización, análisis y seguridad.
- [hacking_tools.md](hacking_tools.md): herramientas conocidas, casos de uso y contexto de uso.
- [owasp.md](owasp.md): base para vulnerabilidades web, OWASP Top 10 y mitigaciones.

## Criterios de contenido

- Español técnico, manteniendo nombres de herramientas y términos estándar en inglés.
- Nivel avanzado, pero explicado de forma directa para aprender desde cero.
- Ejemplos prácticos con salida esperada cuando sea útil.
- Escenarios reales orientados a ataque y defensa.
- Referencias a OWASP Top 10 y CVE cuando aporten contexto.
